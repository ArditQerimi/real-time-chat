# Real-Time Chat App

Simple chat application with rooms and real-time messages.

## What I used

- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express + Typescript + Socket.io
- Database: Postgresql
- Messages saved in: Redis
- Runs with: Docker

## Main Features

- Register/Login
- Create or join chat rooms
- Send & receive messages instantly in group - general channel
- Send & receive messages instantly in private
- See when someone is typing

## How to Run (easiest way)

```bash
# 1. Go to project folder
cd chat-app

# 2. Start everything with one command
docker compose up --build
```
